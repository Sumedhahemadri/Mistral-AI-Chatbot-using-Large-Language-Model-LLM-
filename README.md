🤖 Mistral AI Chatbot using Mistral-7B LLM

📌 Overview

This project is an AI-powered chatbot built using the Mistral-7B Large Language Model (LLM). It generates human-like responses to user queries through an interactive web interface developed with Gradio. The model is loaded locally using ctransformers and the GGUF model format for efficient inference.
 

 🚀 Features

* AI-powered conversational chatbot
* Real-time text generation
* Interactive web interface using Gradio
* Powered by the Mistral-7B Large Language Model
* Efficient GGUF model loading with ctransformers
* Easy to customize and extend

  

🛠️ Technologies Used

* Python
* Mistral-7B
* Large Language Models (LLM)
* Gradio
* ctransformers
* GGUF
* Prompt Engineering

 ⚙️ Installation
 Clone the repository
 
git clone https://github.com/Sumedhahemadri/Mistral-AI-Chatbot.git
 

  Navigate to the project directory

 
cd Mistral-AI-Chatbot


  Install the required libraries
 
pip install -r requirements.txt
 
 

  ▶️ Running the Project

Run the chatbot:

 
python chatbot.py
